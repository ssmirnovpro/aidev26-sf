### What is Temporal?

Temporal is an open-source Durable Execution platform that lets you build reliable workflows for AI agents. The server maintains state and ensures workflow execution, while Workers handle the business logic.

### How It Works (from materials)

1. Break your application into discrete steps (activities)
2. Use Temporal's API for orchestration and persistence
3. Run Workers to execute your activities

### Why Temporal?

- **Durable execution** — ensures processes can recover from failures and run to completion
- **Scalable architecture** — horizontal scaling built in
- **Flexible primitives** — adaptable building blocks
- **Integrations**: OpenAI Agents SDK (verified), with support for various cloud platforms

## Business Model

**Model: Open Source + Managed Cloud (SaaS)**

| Option | Description | Price |
|--------|-------------|-------|
| **Temporal OSS** | Self-hosted, open-source | Free |
| **Temporal Cloud Essentials** | Managed cloud | from $100/mo |
| **Temporal Cloud (consumption)** | Beyond plan limits | Pay-as-you-go |
| **Enterprise** | Custom SLAs, support | By request |

**Billing units:** Actions (operations between your app and Temporal Cloud), Capacity, Storage, Support.

**Startup program:** $6,000 in free Temporal Cloud credits for startups with funding <$30M and an active Temporal Cloud account with at least one Workflow run.

**Revenue streams:**
- Temporal Cloud subscriptions and consumption
- Enterprise contracts
- Support and SLAs
