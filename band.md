## Product

**BAND** is an enterprise-grade interaction infrastructure for distributed AI agents with embedded governance.

### Key Messages (from booth materials)

> "Let your agents collaborate. Keep them under control."

> "All agents, now in sync."

> "Stop duct-taping. Start scaling."

### Two-Layer Architecture

#### Layer 1: The Agentic Mesh (Collaboration Infrastructure)

- **Agent discovery** — find and connect agents
- **Structured delegation** — organized task handoffs
- **Context & memory** — shared context and memory
- **Message routing** — route messages between agents
- **Human in the loop** — include humans in the workflow
- **Persistent processes** — crash recovery, delivery tracking, and long-running coordination support

Full-duplex, multi-peer communication. A group of agents (e.g., planning, coding, and QA agents) works in a shared "room" with synchronized context.

#### Layer 2: The Agent Interaction Control Plane (Governance Infrastructure)

- **See everything** — complete observability
- **Apply policy at runtime** — enforce policies in real-time
- **Define boundaries** — set interaction boundaries
- **Full traceability** — end-to-end tracking

### What BAND Delivers (from materials)

- Unified governance environment for all agents
- Ship faster — accelerate deployment
- Run reliably — dependable execution
- Govern at runtime — real-time control

### Target Users (from materials)

- **Developers** — tools for building multi-agent systems
- **R&D Teams** — testing and experiment management
- **Enterprise IT** — governance and access control

### Technical Differentiator

BAND **doesn't use LLMs for message routing** (deterministic routing instead). This is a deliberate design choice: using LLMs for routing would introduce the same non-deterministic errors the platform aims to solve.

### Compatibility

**Native adapters (9+):** LangGraph, Claude, OpenClaw, Parlant, Letta, Google, n8n, CrewAI

**Additional integrations:** LangChain, third-party SaaS agents, coding agents (Claude Code, Codex), personal AI assistants

**Protocols:** Python/TypeScript SDKs, Model Context Protocol (MCP), Agent-to-Agent (A2A)
