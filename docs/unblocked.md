# Unblocked

## Product

**Unblocked Context Engine** is a platform that provides AI coding agents with decision-grade context pulled from all engineering sources.

### Key Messages (from materials)

> "Context that saves you time and tokens."

> "Real results from real teams."

### How the Context Engine Works (from materials)

1. **Sources** — connect all engineering data sources
2. **Process** — process and index context
3. **Retrieve** — extract relevant context
4. **Deliver** — deliver context to AI agents

### Features

#### Give Agents the Context Your Best Engineers Have
- Access to complete engineering context: code, PR history, documentation, discussions

#### Improve Code Quality
- Detect potential bugs, logic errors, security vulnerabilities
- Follow best practices using context from Slack discussions, specs, and similar PRs

#### Reduce rework
- Cut down on rework by getting the right context the first time

#### Save tokens
- Save tokens through more precise context

### All Your Context, One Connected Engine

Data sources:
- **Code**: GitHub, GitLab, Bitbucket, Azure DevOps (code and PR history)
- **Tickets**: Jira, Linear, Asana (tickets and epics)
- **Communication**: Slack, Microsoft Teams (decisions and discussions)
- **Documentation**: Confluence, Notion, Google Drive, Coda
- **Q&A**: Stack Overflow for Teams
- **CI**: Buildkite, CircleCI, GitHub Actions, Jenkins

### Enterprise Ready (from materials)

- SOC 2 Type II compliant
- SSO / SCIM support
- Fine-grained data isolation
- Team-based access controls
- Encryption at rest and in transit
- Third-party penetration tests

## Analysis

Unblocked addresses a fundamental challenge with AI coding agents: lack of context. Without understanding architecture, team decisions, and codebase history, AI agents generate code that technically works but may not match team standards. Unblocked's approach combines retrieval with cross-source synthesis across code, documentation, and communication channels.
