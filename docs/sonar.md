# Sonar (SonarQube / SonarSource)

## About the Company

**SonarSource** develops SonarQube and SonarCloud, the leading platforms for static code analysis and code quality management. The company positions itself as "Code Verification for the AI Era" and is headquartered in Geneva, Switzerland.

The company's core concept is **AC/DC (Agent Centric Development Cycle)** — a new development paradigm that puts AI agents at the center of the software development lifecycle.

## Product

**SonarQube Agentic Analysis** is a service that integrates SonarQube's proven analysis capabilities directly into AI coding agent workflows.

### Key Message

> "Verify your code at agent speed"

### Agentic Analysis Capabilities

- Verifies AI-generated code with the same accuracy as a full CI scan, but in seconds instead of minutes
- Automatically checks code against SonarQube quality profiles using complete project context
- Leverages context from the latest SonarQube CI analysis with on-demand restoration for single-file and multi-file analysis
- Closes the Guide-and-Verify loop in real-time

### Compatibility

Works with most AI-powered IDEs and CLIs:
- Claude Code
- Cursor
- Codex
- Gemini CLI
- VS Code with Copilot

### SonarQube MCP Server

A critical bridge connecting SonarQube's analysis engine directly to AI agents, enabling tools to verify code security in real-time.

## The AC/DC Concept

SonarSource introduced the **"Agent Centric Development Cycle" (AC/DC)** concept — a new development paradigm where AI agents take center stage in the software creation cycle, and code verification becomes an automatic, continuous process.

## Analysis

Sonar's positioned itself strategically in the AI development stack as the "guard rail" for AI-generated code. As vibe coding and automated agent-based code generation grow, the need for verification scales proportionally. The SonarQube MCP Server lets agents verify code in real-time — that's a key competitive advantage.
