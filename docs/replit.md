## Product

### Replit — AI-Powered App Creation Platform

A platform that lets you build full-fledged applications using natural language prompts:

- **Replit Agent** — An AI agent that constructs complete applications from text descriptions
- **Agent 4** (current generation) — 10x faster than the previous version, features a design-canvas interface and parallel agent workflows (multiple agents working on the same project simultaneously)
- Browser-based IDE with instant launch
- One-click deployment
- Multiplayer collaboration

### Key Features

- Complete lifecycle: from idea → AI generation → deployment
- Support for 50+ programming languages
- Built-in hosting and databases
- GitHub integration

## Business Model

**Model: Freemium + Consumption-based (Pay-as-you-go)**

| Tier | Description |
|------|-------------|
| **Free** | Basic access, limited compute |
| **Core** | Subscription with increased limits |
| **Pro** | For teams and heavy users (Teams tier being replaced by Pro) |
| **Enterprise** | Custom terms |

**Model Transformation:**
- Pre-2024: Primarily subscription-based (low ARPU)
- Late 2024 onwards: Introduction of **usage-based pricing for AI agents** — dramatically expanding average revenue per user
- Consumption-based stream from AI Agent has become the primary growth driver

**Revenue Sources:**
- AI Agent consumption (primary, new)
- Subscriptions (baseline)
- Enterprise contracts (around 80% gross margins)
- Compute resources

### What Customers Should Expect

| Tier | Price/month | What's Included |
|------|-------------|-----------------|
| **Free** | $0 | Limited compute, basic AI |
| **Core** | $20–$25 | AI credits, more compute |
| **Pro** | $100 | Up to 15 builders, priority support, rollover credits |
| **Enterprise** | On request | Custom terms |

**Pricing structure:**
- Usage-based AI Agent billing (credits consumed)
- Monthly credit allocations with plan
- Pro: rollover credits, up to 15 builders
- Credits burn quickly with frequent Agent use

Actual costs depend on Agent usage patterns. Use Replit's pricing calculator for estimates.

## Analysis

Replit is one of the most explosive growth stories in the AI industry: from $2.8M ARR to $150M in a single year. The company competes directly with Cursor and OpenAI Codex, but distinguishes itself with a full-stack approach (from IDE to deployment). The $9B valuation on $150M ARR reflects extreme market expectations. The goal of $1B ARR by end of 2026 is ambitious but backed by strong growth trajectory.
