# Datadog

## Company Overview

**Datadog** (NASDAQ: DDOG) is a leading cloud-native monitoring and observability platform. The company's increasingly making its mark in AI, rolling out specialized tools for monitoring LLMs and AI agents.

## Products

### 1. Datadog LLM Observability (from conference materials)

> "Iterate, evaluate, and ship more reliable AI agents faster."

A comprehensive observability platform for AI/LLM applications:

- **Monitor agent behavior and performance** — track how your agents are actually behaving in production
- **Evaluate output quality, security, and safety** — ensure your outputs meet quality and security standards
- **Correlate behavior with back-end services and user sessions** — connect agent behavior to backend services and user activity
- **Experiment to balance quality and cost** — run experiments to optimize the quality-to-cost ratio

#### Key Features (as of June 2025)

- **AI Agent Monitoring** — interactive maps showing agent decision paths: inputs, tool calls, calls to other agents, and outputs. Drill down into latency issues, failed tool calls, and infinite loops
- **LLM Experiments** — compare different models, prompts, and configurations side-by-side
- **AI Agents Console** (Preview) — monitoring interface for agentic tools, collecting logs/metrics and viewing telemetry in real time
- End-to-end tracing: inputs, outputs, latency, token usage, errors

### 2. Datadog for Startups (from conference materials)

> "Build, Scale, and Optimize"

A program designed for startups navigating the AI era:

- **Up to $100,000 USD** in credits (for early-stage startups) for the first year
- **Core Services**: Infrastructure Monitoring, Log Storage Monitoring
- **Advanced Capabilities**: Error Tracking, LLM Observability
- **Reliability & Performance**: Real-time visibility across your stack
- **Community-Driven**: access to Datadog's startup community

#### Value Proposition

- **Build Quickly**: accelerate development with AI-powered insights
- **Monitor Confidently**: full-stack monitoring out of the box
- **Scale Seamlessly**: maintain visibility as you grow

## Business Model

**Model: Usage-based SaaS subscription**

| Tier | Price | Notes |
|-------|-----------|------------|
| **Free** | $0 | Up to 5 hosts |
| **Pro** | from $15/host/month | Infrastructure monitoring |
| **Enterprise** | from $23/host/month | Advanced features |
| **DevSecOps Pro** | from $22/host/month | Security focus |
| **DevSecOps Enterprise** | $34/host/month | Full security suite |

**Add-on Products:**
- APM: from $31/host/month
- LLM Observability: new pricing takes effect May 1, 2026

**Pricing Notes:**
- Hourly metering of hosts and custom metrics
- Billable host counts use the high-water mark of the lower 99% of hourly usage (excludes top 1% to reduce spike impact)

**Startup Program:** up to $100,000 in credits for the first year
