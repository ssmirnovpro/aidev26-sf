# LangChain

## About the Company

**LangChain** is an infrastructure platform for building AI agents. Founded by Harrison Chase, the company's core is the open-source LangChain framework, which has become the de facto standard for developing LLM applications. 35% of Fortune 500 companies use LangChain products.

## Products

### LangChain (Open Source)

A framework for building LLM applications and AI agents. 90 million monthly downloads (LangChain + LangGraph combined).

### LangGraph

An orchestrator for custom agents:
- Low-level control over each agent step
- Memory and human-in-the-loop capabilities
- Durable execution for long-running tasks
- Parallel agent workflows

### LangSmith

A SaaS platform for agent engineering:
- **Observability** — see how agents think and act
- **Evaluation** — test and assess agent behavior
- **Deployment** — deploy agents with one click
- Production data tracing for continuous improvement

## Business Model

**Model: Open Source + SaaS (Freemium + Usage-based)**

| Product | Type | Price |
|---------|------|-------|
| **LangChain** | Open-source framework | Free |
| **LangGraph** | Open-source orchestrator | Free |
| **LangSmith Developer** | SaaS | Free (5K traces/month) |
| **LangSmith Plus** | SaaS | $39/user/month (10K traces) |
| **LangSmith Enterprise (Cloud)** | SaaS | Custom pricing |
| **LangSmith Enterprise (Self-hosted)** | On-prem | Premium (for regulated industries) |

**Pricing:** Usage-based (per API call) + seat-based (for teams).

**Revenue sources:**
- LangSmith SaaS subscriptions (primary)
- Enterprise contracts (self-hosted for finance, healthcare)
- Additional traces above plan limits

### What Customers Should Expect to Pay

| Profile | Typical monthly spend |
|---------|----------------------|
| Individual developer | $0 (5K traces/month) |
| Small team (3–5 people, Plus) | $120–$200 + overage |
| Growing team (10 people, 500K traces) | $400–$600 |
| Mid-size project (3M traces/month, Plus) | ~$1,500 overage alone |
| Enterprise | Custom, contact sales |

- Overage (Plus): $2.50 per 1,000 additional traces (above 10K)
- High-volume users may benefit from Enterprise custom pricing
- Self-hosted Enterprise available for regulated industries (finance, healthcare)
