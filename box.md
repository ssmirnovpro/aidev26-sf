# Box

## About the Company

**Box** is a cloud content management platform built for enterprises. Founded in 2005 by Aaron Levie and Dylan Smith (with co-founders Sam Ghods and Jeff Queisser), the company went public in 2015 and trades on the NYSE under the ticker BOX. Headquartered in Redwood City, California, Box serves 97,000 companies—including 68% of the Fortune 500—for secure file storage, collaboration, and content management. Unlike consumer-focused products like Dropbox or Google Drive, Box has evolved into an enterprise-focused platform with heavy emphasis on security, compliance, and integration with enterprise systems.

## Product

**Box** is an enterprise content management platform that brings together cloud storage, collaboration, e-signature, workflow automation, and AI capabilities.

### Box AI

AI features built directly into the Box platform, powered by models from OpenAI, Google, and Anthropic:

- **Box Agent** (GA as of April 2026) — an AI agent that takes natural language instructions and autonomously handles complex tasks: searching company files, analyzing and synthesizing data, creating new content. Works in three modes:
  - **Standard Mode** — everyday tasks: finding information, summarizing documents, synthesizing data
  - **Pro Mode** — for tasks requiring deep analysis, planning, and iterative refinement (Enterprise Advanced)
  - **Expanded Mode** — increased context and query limits for high-volume, mission-critical processes (Enterprise Advanced)

- **Box AI Studio** — a no-code tool for admins to build custom AI agents for specific workflows: configure behavior, choose models, define knowledge bases and rules (Enterprise Advanced only)

- **Box Extract** — pulls structured data from unstructured content (contracts, invoices, forms) using generative AI

- **Box AI Home** — a unified interface for working with Box's AI features

### Box MCP Server

A server based on the Model Context Protocol (a standard proposed by Anthropic) that enables secure connections between third-party AI tools and content in Box:

- **Remote clients:** Claude, Copilot Studio, ChatGPT, GitHub Copilot, Figma, Mistral Le Chat, OpenAI AgentKit, Anthropic Messages API, Azure API Center, Microsoft Foundry Tools, Groq, LangSmith Agent Builder, Amazon Quick Suite, Windows
- **Self-hosted setup:** Cursor, Claude Desktop
- Supports advanced search, multi-file analysis, metadata extraction
- Enforces corporate security policies and access controls

### Box Automate

A platform for orchestrating agentic workflows (GA as of April 2026):

- No-code drag-and-drop workflow builder
- Routes tasks between humans, Box agents, and external systems
- Integrates with Box AI, Box Extract, Box Forms, Box DocGen, Box Hubs, Box Sign, Box AI Studio, Box Apps

### Other Platform Capabilities

- **Box Shield** — data loss prevention, threat detection
- **Box Governance** — content lifecycle management, retention policies
- **Box Sign** — e-signature
- **Box Hubs** — content publishing and distribution
- **Box DocGen** — template-based document generation

## Business Model

**Model: SaaS subscription (per user/month)**

Box makes money through corporate subscriptions to its cloud content management platform. It serves 97,000 companies, including 68% of the Fortune 500.

### Pricing Plans

| Plan | Price (annual billing) | Key Features |
|------|------------------------|--------------|
| **Business** | from $15/user/month | Basic cloud storage, collaboration |
| **Business Plus** | $25/user/month | Unlimited storage, advanced security, eDiscovery |
| **Enterprise** | $35/user/month | Unlimited storage, files up to 50GB, advanced admin controls |
| **Enterprise Plus** | $50/user/month | Full AI suite (Box Agent, Box AI Studio), Box Shield, Box Governance |
| **Enterprise Advanced** | Contact sales | Everything in Enterprise Plus + Pro/Expanded modes for Box Agent, full automation via Box Automate |

### Additional Details

- Per-seat pricing with multi-year contracts
- Upselling AI features (Box AI) as a driver for ARPU growth
- Box Automate available for Business plans and above; full agentic workflow automation suite requires Enterprise Advanced
- Consulting, integrations, and professional services
